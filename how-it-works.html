How Ghar Doctor Works

From wearable sensor to real-time dashboard — here’s how your health data travels securely and instantly.

Vitals Detection

Body sensors continuously monitor heart rate & temperature with medical-grade precision.

ESP32 Processing

ESP32 microcontroller processes raw sensor data and prepares it for secure transmission.

WiFi Transmission

Data is transmitted in real-time via encrypted WiFi channels to cloud infrastructure.

Cloud Storage

Encrypted medical data is stored securely for instant access and future analysis.

Live Dashboard

Doctors & patients can track vitals live using a responsive web dashboard.
